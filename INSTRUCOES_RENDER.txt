PASSO A PASSO PARA SUBIR NO RENDER

1) Suba esta pasta no GitHub:
   git add .
   git commit -m "Versao limpa para Render"
   git push origin main

2) No Render, configure:
   Build Command: npm install
   Start Command: npm start

3) Variaveis de ambiente obrigatorias:
   DATABASE_URL = URL do PostgreSQL do Render
   JWT_SECRET = qualquer senha grande/aleatoria
   NODE_ENV = production

4) O arquivo aberto pelo sistema e:
   public/index.html

5) Se aparecer versao antiga no navegador:
   aperte Ctrl + F5 ou abra em guia anonima.

PUSH NOTIFICATIONS / PWA
1. Gere as chaves VAPID localmente ou no shell do Render:
   node -e "const webpush=require('web-push'); console.log(webpush.generateVAPIDKeys())"
2. No Render, adicione estas variáveis de ambiente:
   VAPID_PUBLIC_KEY=<publicKey gerada>
   VAPID_PRIVATE_KEY=<privateKey gerada>
   VAPID_SUBJECT=mailto:seuemail@seudominio.com.br
3. Faça novo deploy.
4. No sistema, entre logado e clique em Notificações > Receber Push no Celular.
5. Permita a notificação no navegador. O dispositivo ficará cadastrado no banco.
